ESTENSIONE DEI TEMPLATE TWIG

    - TEMPLATE BASE:
    <html>
        <body>
            {% block 'percoso file che estende' %}
            {% endblock %}
        </body>
    </html>


    - TEMPLATE ESTENSIONE:
    {% extends 'percorso file da estendere' %}
    {% block main %}
        ...
        ...
    {% endblock %}
